| 図1 | ボールの軌跡は上に凸の放物線である。また、Aさんのいる位置（始点）の高さが、Bさんのいる位置（終点）の高さよりも高いことを示している。 |
| Aさんが投げた瞬間のボールの高さの方が、Bさんに届く直前のボールの高さより高いとき | 始点の高さを $h\_A$、終点の高さを $h\_B$ とすると、$h\_A \> h\_B$ である。ボールは位置エネルギーが減少する運動をしている。 |
| 力学的エネルギー保存則 | 始点と終点での力学的エネルギーが等しいので、$\\frac{1}{2}mv\_A^2 \+ mgh\_A \= \\frac{1}{2}mv\_B^2 \+ mgh\_B$ が成り立つ。 |
| エネルギー保存則と高さの条件の組み合わせ | $h\_A \> h\_B$ より $mgh\_A \> mgh\_B$ なので、エネルギー保存則の式から $\\frac{1}{2}mv\_A^2 \< \\frac{1}{2}mv\_B^2$ が導かれる。速さは正の値なので、$v\_A \< v\_B$ となる。 |
| 水平速度一定の式と速さの大小関係の組み合わせ | $v\_A \\cos\\theta\_A \= v\_B \\cos\\theta\_B$ と $v\_A \< v\_B$ から、$\\cos\\theta\_A \= \\frac{v\_B}{v\_A}\\cos\\theta\_B$ であり、$\\frac{v\_B}{v\_A} \> 1$ なので、$\\cos\\theta\_A \> \\cos\\theta\_B$ となる。 |
| 三角関数の性質 | 角度 $\\theta$ が $0^\\circ \< \\theta \< 90^\\circ$ の範囲では、$\\cos\\theta$ は単調に減少する関数である。 |
| $\\cos\\theta\_A \> \\cos\\theta\_B$ から導かれる角度の関係 | $\\cos$が単調減少関数であることから、$\\theta\_A \< \\theta\_B$ が導かれる。 |
| Aさんが投げた瞬間のボールの高さと...高さが等しい場合には、$v\_A=v\_B$, $\\theta\_A=\\theta\_B$ | この記述は、高さが同じ（$h\_A=h\_B$）という特殊な場合の答えを示している。この条件から $v\_A=v\_B$, $\\theta\_A=\\theta\_B$ となることを利用して、高さが異なる本問の状況を類推・検証するためのヒントとなる。 |
