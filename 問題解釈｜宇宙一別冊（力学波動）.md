## 1

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| $v-t$ グラフ | 縦軸に速度 $v$ \[m/s\]、横軸に時刻 $t$ \[s\] をとる。グラフの**傾きが加速度 $a$ \[m/s²\]** を表し、グラフの線と $t$ 軸で囲まれた面積が移動距離（変位）を表す。 |
| $t \= 0\\text{ s} \\sim 5.0\\text{ s}$ の間 | グラフの描画範囲（定義域）は $0 \\le t \\le 5.0$ である。最終的な時刻 $t \= 5.0$ における速度 $v$ を計算し、終点としてプロットする必要がある。 |
| 加速度が一定（等加速度直線運動） | 文中で加速度の値が一定値で与えられているため、グラフは**折れ線、または直線**（1次関数 $v \= v\_0 \+ at$）で構成される。 |
| (1) 静止している | 時刻 $t \= 0$ における速度（初速度）が $v\_0 \= 0\\text{ m/s}$ である。グラフは必ず**原点 $(0, 0)$ を通る**。 |
| (1) 加速度 $2.0\\text{ m/s}^2$ | 直線の傾きが $+2.0$ である。$t \= 5.0$ のとき、速度は $v \= 0 \+ 2.0 \\times 5.0 \= 10\\text{ m/s}$ となる。 |
| (2) 初速度 $4.0\\text{ m/s}$ | 時刻 $t \= 0$ において $v \= 4.0$。$v$ 軸（縦軸）の**切片が $4.0$** となる。 |
| (2) 最初の2.0秒間は加速度 $3.0\\text{ m/s}^2$ | $0 \\le t \\le 2.0$ の範囲で傾き $3.0$ の直線。$t \= 2.0$ での到達速度は $v \= 4.0 \+ 3.0 \\times 2.0 \= 10\\text{ m/s}$ である。 |
| (2) その後3.0秒間 | 時間の範囲は $2.0 \\le t \\le 5.0$ である（$2.0 \+ 3.0 \= 5.0$）。 |
| (2) 加速度を与えず | **加速度 $a \= 0$**、すなわち等速度運動である。グラフは **$t$ 軸に平行な水平線**（$v \= 10$ で一定）となる。 |
| (3) 初速度 $12\\text{ m/s}$ | 時刻 $t \= 0$ において $v \= 12$。$v$ 軸の切片が $12$ となる。 |
| (3) 進行方向と逆向きに加速度 $4.0\\text{ m/s}^2$ | 初速度の向きを正とすると、\*\*加速度は負（$a \= \-4.0\\text{ m/s}^2$）\*\*となる。グラフは右下がりの直線（負の傾き）を描く。 |
| (3) 5.0秒間与えた | $t \= 5.0$ での速度を計算すると $v \= 12 \- 4.0 \\times 5.0 \= \-8.0\\text{ m/s}$。速度が負になるため、グラフは **$t$ 軸（$v=0$）を突き抜けて下側まで描く**必要がある。 |
| (3) （計算で導く点） | 直線が $t$ 軸を横切る（一瞬静止する）時刻は $t \= 12 / 4.0 \= 3.0\\text{ s}$。作図の際、**$(3.0, 0)$ を正確に通る**ように描かなければならない。 |

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| $v-t$ グラフ（図2） | グラフの**傾きが加速度 $a$** \[m/s²\] を表し、グラフの線と横軸（$t$ 軸）で囲まれた\*\*面積が変位（位置の変化量）\*\*を表す。 |
| $t \= 0\\text{ s}$ で $x \= 0\\text{ m}$ の位置 | 初期位置が原点である。これにより、ある時刻までの\*\*「面積（変位）」を計算した値が、そのままその時刻の「位置 $x$」\*\*の値となる。 |
| グラフが直線で構成されている | 各区間において加速度が一定（等加速度直線運動、または等速度運動）であることを意味する。面積計算には三角形や台形の公式がそのまま使える。 |
| $0 \\le t \\le 2.0\\text{ s}$ の加速度 | $(t, v) \= (0, 0)$ と $(2.0, 8.0)$ の2点を通る直線の傾きを求める。式は $a \= \\frac{8.0 \- 0}{2.0 \- 0} \= 4.0\\text{ m/s}^2$ となる。 |
| $2.0 \\le t \\le 4.0\\text{ s}$ の形状 | 速度 $v \= 8.0\\text{ m/s}$ で一定の「等速度運動」である。この区間の変位は長方形の面積として計算できる。 |
| $t \= 6.0\\text{ s}$ における位置 | $t \= 0 \\sim 6.0$ の範囲は常に $v \\ge 0$（$x$ 軸の正の向きに移動）である。この範囲の**台形の面積**を求めることで位置が特定できる。 |
| $t \= 6.0\\text{ s}$ でグラフが $t$ 軸と交わる | この瞬間に物体の**速度が $0$ になり、運動の向きが反転する**（折り返し点）。これ以降、位置 $x$ の値は減少に転じる。 |
| $6.0 \< t \\le 8.0\\text{ s}$ の領域 | 速度 $v$ が負の値をとる。これは物体が **$x$ 軸の負の向き（逆向き）に運動**していることを表す。 |
| $t \= 8.0\\text{ s}$ における位置 | $t \= 0 \\sim 6.0$ で進んだ距離（正の面積）から、$t \= 6.0 \\sim 8.0$ で戻った距離（$t$ 軸より下の三角形の面積）を**差し引く**ことで求まる。 |
| グラフの対称性と目盛り | $t \= 7.0$ で $v \= \-4.0$、 $t \= 8.0$ で $v \= 0$ と読み取れる。$t \= 6.0 \\sim 8.0$ の区間は、底辺 $2.0$、高さ $4.0$ の三角形として面積計算を行う。 |

## 3

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 等加速度運動の公式を利用して | 基本3公式（$v \= v\_0 \+ at$ , $x \= v\_0t \+ \\frac{1}{2}at^2$ , $v^2 \- v\_0^2 \= 2ax$）を、与えられた変数（$v\_0, a, t, x, v$）に応じて使い分ける必要がある。 |
| 最初に進行している方向を正の方向とする | 初速度 $v\_0$ の値を正（$+$）として基準にする。これ以降、向きが指定される加速度や変位などは、この符号規則に従って立式しなければならない。 |
| (1) $t=0 \\text{ s}$ において $3.0 \\text{ m/s}$ | 初速度 $v\_0 \= \+3.0 \\text{ m/s}$ である。 |
| (1) 進行方向と同じ向きに加速度 $2.0 \\text{ m/s}^2$ | 加速度 **$a \= \+2.0 \\text{ m/s}^2$** である。符号が正のため、時間の経過とともに速度は増加し続ける。 |
| (1) 変位を求めよ | $t=7.0 \\text{ s}$ における位置の変化量を求める。公式 $x \= v\_0t \+ \\frac{1}{2}at^2$ に各値を代入して計算する。 |
| (2) $t=0 \\text{ s}$ において $10 \\text{ m/s}$ | 初速度 $v\_0 \= \+10 \\text{ m/s}$ である。 |
| (2) 進行方向と逆向きに加速度 $4.0 \\text{ m/s}^2$ | 加速度 **$a \= \-4.0 \\text{ m/s}^2$** と、負の符号をつけて代入しなければならない。 |
| (2) 速度を求めよ | $v \= 10 \- 4.0 \\times 5.0 \= \-10 \\text{ m/s}$ となる。計算結果に負の符号がつく場合、それは「逆向きに $10 \\text{ m/s}$」であることを意味する。 |
| (2) 変位を求めよ | 公式で求まるのは「道のり（移動距離の総和）」ではなく、\*\*「初期位置からの位置の差」\*\*である。この問題では途中で速度が $0$ になり折り返すため、計算結果が $0$ になれば「元の位置に戻った」ことを示す。 |
| (3) $6.0 \\text{ m/s}$ で進んでいる | 初速度 $v\_0 \= \+6.0 \\text{ m/s}$ である。 |
| (3) $8.0 \\text{ m}$ 進んだとき | 変位 $x \= \+8.0 \\text{ m}$ である。 |
| (3) 速度は $10 \\text{ m/s}$ になった | 変化後の速度 $v \= \+10 \\text{ m/s}$ である。 |
| (3) （公式の選択） | **時刻 $t$ の情報が与えられていない**ため、時間を消去した公式 **$v^2 \- v\_0^2 \= 2ax$** を用いるのが最も効率的である。 |
| (3) 加速度の大きさ | 加速度の絶対値を求める。「進行方向と同じ向き」という記述から、符号は正のままでよい。 |

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 斜め方向に発射（斜方投射） | 運動を\*\*「水平方向（等速直線運動）」と「鉛直方向（鉛直投げ上げ運動）」の2軸に分解\*\*して考える。 |
| 初速度 $196 \\text{ m/s}$ 、角度 $30^\\circ$ | 初速度を成分分解する。水平初速度 $v\_{x0} \= 196 \\cos 30^\\circ \= 98\\sqrt{3} \\text{ m/s}$、鉛直初速度 $v\_{y0} \= 196 \\sin 30^\\circ \= 98 \\text{ m/s}$ となる。 |
| 水平な地面から発射 | 投射点と落下点の高さが等しい。投げ上げてから最高点に達するまでの時間と、最高点から地面に戻るまでの**時間が等しい（対称性）**。 |
| 鉛直方向の加速度 | 鉛直上向きを正とすると、加速度は重力加速度 $g$ を用いて **$a\_y \= \-9.8 \\text{ m/s}^2$** となる。 |
| (1) 最高点の高さ | 最高点では**鉛直方向の速度成分が $v\_y \= 0$** になる。この条件を用いて最高点に達する時刻 $t\_1$、または公式 $v\_y^2 \- v\_{y0}^2 \= 2ay$ を使って高さを求める。 |
| (2) 何秒後に地上に落ちるか | 落下時の条件は **鉛直方向の変位 $y \= 0$** である。または最高点までの時間 $t\_1$ を2倍することで全滞空時間を求められる。 |
| (3) 何m離れた位置に | 水平方向の移動距離（射程）を求める。水平方向は加速度 $a\_x \= 0$ のため、式は **$x \= v\_{x0} \\times (\\text{全滞空時間})$** となる。 |
| $\\sqrt{3} \= 1.7$ とする | 計算の最終段階でこの近似値を代入する。途中の計算では $\\sqrt{3}$ のまま保持したほうが誤差が出にくい。 |
| 数値の計算のコツ | 初速度の鉛直成分 $98$ は重力加速度 $9.8$ のちょうど10倍になっている。これに気づくと最高点までの時間（$10\\text{ s}$）などが暗算で導ける。 |
| 有効数字 | 与えられた数値 $196$、$9.8$ がいずれも3桁（$9.8$ は $9.80$ とみなすのが通例）であり、$\\sqrt{3}=1.7$ が2桁であることから、最終的な解答は**有効数字2桁**でまとめるのが適切である。 |

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 同方向に……一直線上を走行 | 1次元（直線）上の運動である。進行方向を正（$+$）とすると、$v\_A \= \+20 \\text{ m/s}$、$v\_B \= \+26 \\text{ m/s}$ と符号を統一して扱える。 |
| $t=0 \\text{ s}$ で、車間距離は $42 \\text{ m}$ | 初期状態での位置関係を表す。Bの位置を原点 $x\_B \= 0$ とすると、Aの位置は $x\_A \= 42$ と設定できる。 |
| (1) 自動車Aに対する自動車Bの速度 | \*\*「相対速度」**の定義。**「（相手の速度）$-$（自分の速度）」\*\*の公式に当てはめ、$v\_{BA} \= v\_B \- v\_A$ を計算する。これは「Aから見たBの速度」を意味する。 |
| (2) それぞれの速度を保ったまま | \*\*等速度運動（加速度 $a=0$）\*\*であることを示している。時間 $t$ における各車両の位置は $x \= x\_0 \+ vt$ という単純な1次関数で表される。 |
| (2) 2台の車が衝突する | **「2台の位置が一致する（$x\_A \= x\_B$）」**、または\*\*「BがAに追いつく」**という条件。相対速度の視点では、**「Bが相対速度 $v\_{BA}$ で距離 $42 \\text{ m}$ を移動しきった」\*\*と読み替えられる。 |
| 衝突の条件（相対的な視点） | 観測者をAに固定して考えると、Aは静止しているように見え、Bが速度 $v\_{BA}$ で $42 \\text{ m}$ 後方から近づいてくる運動として単純化できる。式は $42 \= v\_{BA} \\times t$ となる。 |
| 図の目盛り（車間距離の定義） | 図中の矢印は「Bの先端」から「Aの後端」までを指している。この $42 \\text{ m}$ がゼロになった瞬間が衝突であり、車の全長を考慮する必要はない（質点として扱ってよい）。 |
| 数値の計算 | $v\_B \> v\_A$ であるため、相対速度 $v\_{BA}$ は正の値となり、時間の経過とともに車間距離が確実に縮まることが確認できる。 |

## 8

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 床の上に置かれた／重ねて……置かれており | 物体は静止している。したがって、物体A、物体Bそれぞれにおいて\*\*「力のつりあい（合力＝0）」\*\*が成立している。 |
| 質量 $m\_1$、質量 $m\_2$ | 各物体の重心に、鉛直下向きの重力 **$m\_1g$** および **$m\_2g$** がそれぞれ作用している。 |
| 物体Aには上から力 $F$ が加えられている | 物体Aに対して、直接的に「下向き」の外部入力としての力が作用している。 |
| 物体Aが物体Bから受ける力 $N\_A$ | 物体AとBの接触面における垂直抗力である。向きは**鉛直上向き**。これと対になって、**物体Bは物体Aから大きさ $N\_A$ の力（作用・反作用）を鉛直下向きに受けている**。 |
| 物体Bが床から受ける力 $N\_B$ | 床と物体Bの接触面における垂直抗力である。向きは**鉛直上向き**。 |
| 鉛直方向の正負の定義 | 立式のために座標軸を決める必要がある。通常、**鉛直上向きを正**として、「（上向きの力の和）＝（下向きの力の和）」という方程式を立てる。 |
| 物体Aについての力のつりあい | Aに働く力は「重力 $m\_1g$（下）」「外力 $F$（下）」「垂直抗力 $N\_A$（上）」の3つである。式は **$N\_A \= m\_1g \+ F$** となる。 |
| 物体Bについての力のつりあい | Bに働く力は「重力 $m\_2g$（下）」「Aから押される力 $N\_A$（下）」「床からの垂直抗力 $N\_B$（上）」の3つである。式は **$N\_B \= m\_2g \+ N\_A$** となる。 |
| 全体を1つの物体とみなす視点 | 物体AとBをひとまとめ（質量 $m\_1+m\_2$）として考えると、外力は「重力 $(m\_1+m\_2)g$（下）」「外力 $F$（下）」「床からの垂直抗力 $N\_B$（上）」となり、**$N\_B \= (m\_1+m\_2)g \+ F$** が導かれる（検算に有効）。 |
| 作用・反作用の法則 | $N\_A$ は「BがAを押し上げる力」であるが、Bのつりあいを考える際には「AがBを押し下げる力」として、同じ大きさ $N\_A$ を用いる必要がある。 |

| 波長 $\\lambda \= 600$ nm | 計算時に単位を統一する必要がある。$600 \\text{ nm} \= 6.0 \\times 10^{-5}$ cm。 $R \= 40$ cm と合わせて計算する。 |
| 垂直入射の仮定 | 図では斜めに描かれていることもあるが、干渉の計算においては光がレンズ上面から垂直に入射し、垂直に反射して戻るものとして光路差 $2d$ を扱う。 |

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 2枚のガラス板（くさび形空気層） | 干渉が起きる場所は「2枚のガラスに挟まれた空気層」である。上のガラスの下面での反射と、下のガラスの上面での反射が干渉する。 |
| 左端だけ接触 | 左端 ($x=0$) では空気層の厚さが $0$ である。接触点では位相が半波長分ズレるため、必ず「暗線」となる。 |
| 角度 $\\theta$ だけ傾ける | 左端から距離 $x$ の地点における空気層の厚さ $d$ は、幾何学的に $d \= x \\tan \\theta$ と表される。 |
| $\\theta$ は十分小さい | 近似式 $\\tan \\theta \\approx \\theta$ （ラジアン）が成立する。したがって、厚さは $d \\approx x\\theta$ と簡略化して計算に用いる。 |
| 真上から波長 $\\lambda$ の光を入射 | 入射光は空気層を往復するため、2つの反射光の幾何学的な経路差は $2d \= 2x\\theta$ となる。 |
| 下のガラス上面での反射 | 屈折率の小さい空気から大きいガラスへの反射（固定端反射）であるため、位相が $\\pi$ ズレる（経路差に換算して $\\lambda/2$ の変化）。 |
| 上のガラス下面での反射 | 屈折率の大きいガラスから小さい空気への反射（自由端反射）であるため、位相の変化は起こらない。 |
| 強め合う条件 (1) | 片方の反射でのみ位相が反転するため、経路差 $2x\\theta$ が半波長の奇数倍のとき強め合う。即ち $2x\\theta \= (m \+ 1/2)\\lambda$ となる。 |
| 隣り合う明線の間隔 $\\Delta x$ (2) | 条件式より $x\_m \= \\frac{(m \+ 1/2)\\lambda}{2\\theta}$。 隣り合う間隔は $\\Delta x \= x\_{m+1} \- x\_m \= \\frac{\\lambda}{2\\theta}$ となり、$\\lambda$ と $\\theta$ で決まる定数（等間隔）である。 |
| 様々な色の光（波長の違い） | 可視光において、赤色の光は波長 $\\lambda$ が長く、紫色の光は波長 $\\lambda$ が短いという物理的性質を利用する。 |
| 色と間隔の関係 (3) | 間隔の式 $\\Delta x \= \\frac{\\lambda}{2\\theta}$ より、波長 $\\lambda$ が長いほど間隔 $\\Delta x$ は広くなる。したがって、赤色に近いほど間隔は広がる。 |
