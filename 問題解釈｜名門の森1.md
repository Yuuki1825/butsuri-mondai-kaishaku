## 力学

### 1 放物運動

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 水平方向に $x$ 軸、鉛直方向に $y$ 軸 | 重力加速度ベクトルは $\\vec{g} \= (0, \-g)$ と表される。水平方向（$x$ 方向）には力が働かない。 |
| 時刻 $t=0$ に原点O $(0,0)$ から...速さ $v\_0$ で投射 | 小球Pの初速度成分は $v\_{Px0} \= v\_0 \\cos\\theta$、 $v\_{Py0} \= v\_0 \\sin\\theta$ である。 |
| 小球Pの運動 | $x$ 方向は等速直線運動（$x\_P \= v\_0 \\cos\\theta \\cdot t$）、 $y$ 方向は初速度 $v\_0 \\sin\\theta$ の鉛直投げ上げ運動（$y\_P \= v\_0 \\sin\\theta \\cdot t \- \\frac{1}{2}gt^2$）である。 |
| 点 $(a, b)$ から小球Qを自由落下させる | 小球Qの初期条件は、位置 $(a, b)$、初速度 $v\_{Q0} \= 0$ である。 |
| 小球Qの運動 | 常に $x\_Q \= a$ であり、 $y$ 方向は自由落下運動（$y\_Q \= b \- \\frac{1}{2}gt^2$）である。 |
| 鉛直線 $(x=a)$ を横切る | 小球Pの $x$ 座標が $a$ になる条件 $x\_P \= a$ から、通過時刻 $t \= \\frac{a}{v\_0 \\cos\\theta}$ が求まる。 |
| 両物体は衝突する | 同一時刻 $t$ において、両物体の座標が一致する（ $x\_P \= x\_Q$ かつ $y\_P \= y\_Q$ ）。 $x\_P \= x\_Q \= a$ は (1) で満たされているので、 $y\_P \= y\_Q$ を立式する。 |
| $v\_0$ の値にかかわらず衝突する | 衝突条件 $y\_P \= y\_Q$ を整理した式から $v\_0$ が消去され、$a, b, \\theta$ のみの関係式（$\\tan\\theta \= \\frac{b}{a}$）が得られることを意味する。これは「Pを投射する瞬間にQをねらっている」という幾何学的な条件（モンキーハンティング）を指す。 |
| $x$ 軸の上側 $(y \\ge 0)$ で衝突 | 衝突時の $y$ 座標が 0 以上であるという条件から、不等式 $y\_P(t) \\ge 0$ （または $y\_Q(t) \\ge 0$ ）を立てて $v\_0$ の範囲を制限する。 |
| Qから見ると | 相対速度 $\\vec{v}\_{PQ} \= \\vec{v}\_P \- \\vec{v}*Q$ を考える。両物体は共に重力加速度 $g$ を受けているため、相対加速度 $\\vec{a}*{PQ} \= \\vec{g} \- \\vec{g} \= 0$ となり、Qから見たPは「等速直線運動」をしているように見える。 |
| 相対運動による衝突時間 $t\_0$ | Qから見たPの初速度（＝相対速度）の大きさは $v\_0$ であり、衝突までの相対的な移動距離は2点間の初期距離 $L \= \\sqrt{a^2 \+ b^2}$ である。等速運動の関係式から $t\_0 \= \\frac{L}{v\_0}$ と求まる。 |

### 3 放物運動

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 小球 P が高さ $h$ だけ静かに落下 | 初速度 $v\_0 \= 0$ の自由落下である。斜面衝突直前の速さは、力学的エネルギー保存則より $v \= \\sqrt{2gh}$ である。 |
| 点 O で斜面と衝突 | 衝突直前の速度ベクトルは鉛直下向きである。これを斜面に平行な成分 $v\_x \= v \\sin \\theta$ と、垂直な成分 $v\_y \= \-v \\cos \\theta$ に分解して考える。 |
| 滑らかな斜面 | 斜面方向の摩擦力は働かない。したがって、衝突前後で斜面に平行な速度成分 $v\_x$ は変化しない。 |
| 反発係数 $e (\<1)$ | 斜面に垂直な速度成分 $v\_y$ は、衝突によって向きが反転し、大きさは $e$ 倍になる。直後の成分は $v\_y' \= ev \\cos \\theta$ となる。 |
| 直後の速度の向きが斜面となす角 $\\alpha$ | $\\tan \\alpha \= \\frac{\\text{垂直成分}}{\\text{平行成分}} \= \\frac{v\_y'}{v\_x}$ より、$\\theta, e$ を用いて立式できる。 |
| 斜面座標系（$x, y$ 軸）における加速度 | 重力加速度 $g$ を成分分解する。$x$ 軸正方向（斜面下向き）の加速度は $a\_x \= g \\sin \\theta$、$y$ 軸正方向（斜面垂直上向き）の加速度は $a\_y \= \-g \\cos \\theta$ である。 |
| 再び斜面上の点 $A\_1$ で衝突 | 放物運動の終点は、$y$ 方向の変位が再び $0$ となる点である。$y \= v\_y' t \+ \\frac{1}{2} a\_y t^2 \= 0$ より、滞空時間 $t$ が求まる。 |
| 距離 $OA\_1$ | 滞空時間 $t$ の間の $x$ 方向の移動距離を求める。$x \= v\_x t \+ \\frac{1}{2} a\_x t^2$ に値を代入する。 |
| $A\_1, A\_2, \\dots$ で次々と衝突をくり返し | 各衝突において $v\_x$ は加速度 $a\_x$ により増加し続けるが、垂直成分 $v\_y$ は衝突のたびに $e$ 倍ずつ減衰していく。 |
| $A\_1 A\_2$ 間を飛ぶ時間 | 次の区間の滞空時間は、衝突直後の $y$ 方向初速度のみに比例する。衝突直前の $v\_y$ の大きさは直後の大きさと等しい（$y$ 方向運動の対称性）ため、滞空時間は公比 $e$ の等比数列となる。 |
| やがて P ははね返らなくなり | 無限回衝突した後の極限状態を考える。滞空時間の総和（無限等比級数の和）は収束し、有限の時間 $T \= \\frac{t\_1}{1-e}$ で点 B に到達する。 |
| 点 B 以後は斜面に沿って滑り降りる | 垂直方向の運動が消滅した状態。距離 $OB$ は、時間 $T$ の間における $x$ 方向の等加速度運動の総変位として計算できる。 |

### 4 剛体のつり合い

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 一様なはしご | はしごの重心はその幾何学的な中点（下端 B から $5l$ の位置）にあり、重力 $Mg$ はそこに働く。 |
| 滑らかな鉛直壁 | 壁からの抗力は垂直抗力のみであり、摩擦力は働かない。したがって、上端 A で受ける抗力の向きは水平右向きである。 |
| 前方 $6l$、長さ $10l$ | はしご、壁、床が作る直角三角形の辺の比は $6:8:10 \= 3:4:5$ である。はしごが床となす角を $\\theta$ とすると $\\cos\\theta=0.6, \\sin\\theta=0.8$ であり、上端 A の高さは $8l$ である。 |
| 立てかけてある（静止） | 剛体のつり合い条件「（水平・鉛直成分ごとの）力の和が $0$」および「任意の点まわりの力のモーメントの和が $0$」が成り立つ。 |
| 静止摩擦係数は $\\frac{1}{2}$ | 床から受ける摩擦力を $f$、垂直抗力を $N\_B$ とすると、はしごが滑らないためには $f \\le \\frac{1}{2}N\_B$（最大摩擦力の不等式）を満たす必要がある。 |
| 下端 B が床から受ける抗力 | 床からの垂直抗力 $N\_B$ と静止摩擦力 $f$ の合力を指す。その大きさは $\\sqrt{N\_B^2 \+ f^2}$ で求められる。 |
| はしごは滑ってしまう | 摩擦力の値が最大摩擦力に達したとき（$f \= \\mu N\_B$）が、滑り出す直前の限界条件となる。 |
| 質量 $5M$ の人が登る | 外力として人の重力 $5Mg$ が加わる。人が下端 B から距離 $x$ だけ登ったとき、モーメントのつり合い式に人の重力による項（腕の長さ $x \\cos\\theta$）を追加して考える。 |
| 作用線の交点 P | 3つの力（壁の抗力、床の抗力、全重力の合力）がつり合って静止している剛体では、それら 3 力の作用線は必ず $1$ 点で交わる。 |
| 交点 P の位置を特定する | 壁の抗力（水平線）と、系全体の重心に働く全重力（鉛直線）の交点として P が定まる。このとき、床の抗力の作用線も必ず P を通る。 |

### 5 剛体のつり合い

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 一様な直方体 M | 重心は幾何学的な中心にある。重力 $Mg$ は底面から高さ $b/2$、底面の中心（側面から距離 $a/2$）の点に作用する。 |
| 斜面平行な力 $F$ を上端に加える | 力 $F$ の作用線は斜面から距離 $b$ の位置にある。モーメントを考える際の腕の長さは $b$ となる。 |
| $\\mu \> \\tan\\theta$ | $F=0$ のとき、斜面下向きの重力成分 $Mg \\sin\\theta$ よりも最大静止摩擦力 $\\mu Mg \\cos\\theta$ の方が大きいため、外力を加えない限り静止し続けるという前提条件。 |
| 滑り始めるための条件 (ア) | 斜面下向きの力の合計（$F \+ Mg \\sin\\theta$）が最大静止摩擦力 $\\mu N$ に達したとき。垂直方向のつり合い $N \= Mg \\cos\\theta$ を代入して $F$ を求める。 |
| 倒れ始めないための条件 (イ) | 斜面からの垂直抗力 $N$ の作用点が、底面の範囲内（$0 \\le x \\le a$）に存在し続けること。 $F$ を大きくしていくと、垂直抗力の作用点は徐々に斜面下側の角へと移動する。 |
| 倒れる寸前の状態 | 垂直抗力 $N$ と摩擦力 $f$ の作用点が、直方体の最も下側の角に一致する。この角のまわりの力のモーメントのつり合いを考える際、抗力によるモーメントは $0$ となる。 |
| モーメントのつり合いの立式 | 下側の角を支点とすると、「$F$ によるモーメント $F \\cdot b$」と「重力の斜面平行成分によるモーメント $Mg \\sin\\theta \\cdot b/2$」の和が、「重力の垂直成分によるモーメント $Mg \\cos\\theta \\cdot a/2$」以下であれば倒れない。 |
| 倒れることなく滑らせる | 「滑り出しに必要な最小の力 $F\_{\\text{slide}}$」が「倒れ出しに必要な最小の力 $F\_{\\text{tip}}$」よりも小さいこと、すなわち $(ア) \< (イ)$ が成立する必要がある。 |
| $\\frac{a}{b} \> \\text{ウ}$ の導出 | 不等式 $(ア) \< (イ)$ を整理し、未知の力 $F$ を消去して、幾何学的形状（アスペクト比 $a/b$）と傾斜角 $\\theta$、摩擦係数 $\\mu$ の関係式を導く。 |

### 6 剛体のつり合い

| 一様な円板、厚さ $d$、密度 $\\rho$ | 円板の質量 $M$ は体積（面積 $\\times$ 厚さ）に比例する。面密度を $\\sigma \= \\rho d$ とすると、質量 $M \= S \\sigma$ と表せる。 |
| 点Aで内接する円形の穴 | 重心計算において、大きな円板（正の質量）から小さな円板（負の質量）を差し引く「重ね合わせの原理」を利用できる。 |

| $y$ 軸方向の重心 $y \= イ$ | 点Aを原点 $(0,0)$ とすると、大きな円（半径 $R$）の中心は $(0, R)$、穴（半径 $R/2$）の中心は $(0, R/2)$ である。重心の定義式 $y\_G \= \\frac{\\sum m\_i y\_i}{\\sum m\_i}$ より $y \= イ$ を算出する。 |
| 同じ3つのばねで支え、円板を水平にさせる | 円板が水平であれば、3つの同一のばねは等しく縮んでいる。したがって、各ばねから受ける上向きの抗力の大きさ $f$ はすべて等しい。 |
| 水平に保つための重心の条件 | 剛体のつり合いより、系全体（円板＋おもり）に働く重力の作用点は、ばねの抗力の合力の作用点と一致しなければならない。等しい3つの平行力の場合、その作用点は「三角形ABCの重心」に一致する。 |
| 点B, C の座標 | 円板の外周を表す方程式 $x^2 \+ (y-R)^2 \= R^2$ と直線 $y \= \\frac{3}{2}R$ の交点として座標 $(x, y)$ を算出する。図の対称性から $x \= \\pm \\frac{\\sqrt{3}}{2}R$ と求まる。 |
| 三角形ABCの重心の位置 | 3頂点 A$(0,0)$, B$(-\\frac{\\sqrt{3}}{2}R, \\frac{3}{2}R)$, C$(\\frac{\\sqrt{3}}{2}R, \\frac{3}{2}R)$ の平均座標として求まる。全重心が位置すべき目標地点は $(0, R)$ となる。 |
| できるだけ軽いおもり | モーメントのつり合いにおいて、目標点 $(0, R)$ から最も遠い位置におもりを置くことで、必要なおもりの質量 $m$ を最小にできる。 |
| おもりを置く位置 (ウ, エ) | 円板単体の重心 $(0, \\frac{7}{6}R)$ は目標点より $y$ 軸正方向（上側）にあるため、おもりは目標点の下側で最も遠い点、すなわち A$(0,0)$ に置くのが最適である。 |
| おもりの質量 $m \= オ$ | 目標点 $(0, R)$ まわりのモーメントのつり合い：(円板の質量 $\\times$ 目標点から円板重心までの距離 $\\frac{1}{6}R$) ＝ (おもりの質量 $m \\times$ 目標点からおもりまでの距離 $R$) より $m$ を算出する。 |

### 7 運動方程式

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 伸び縮みしない一様な綱 | 綱の各部分とおもり P はすべて共通の加速度 $a$ で運動する。また、綱の線密度は一定であり、上端から距離 $x$ までの部分の質量は $m\\frac{x}{l}$、残りの下の部分は $m\\frac{l-x}{l}$ と表せる。 |
| 一定の力 $F$ で引き上げる | 全体を一つの物体（質量 $M+m$）とみなすと、鉛直上向きを正として運動方程式 $(M+m)a \= F \- (M+m)g$ が立てられる。これより、加速度 $a$ は一定（等加速度運動）である。 |
| $F \> (M+m)g$ | 加速度 $a$ が正となり、系が鉛直上向きに加速しながら動き出すことを保証している。 |
| P に働く綱の張力 | おもり P のみに注目した運動方程式 $Ma \= T \- Mg$ を立てることで、張力 $T$ を求められる。 |
| 上端から $x$ のところでの張力 | 綱を $x$ の位置で仮想的に切断し、それより下の部分（質量 $M \+ m\\frac{l-x}{l}$）を一つの物体とみなして運動方程式を立てる。この部分に働く上向きの力はその地点の張力 $T(x)$ である。 |
| 地上で静止していた | 引き上げ開始時（$t=0$）の初速度は $v\_0 \= 0$ である。 |
| 高さ $h$ に達したとき綱からはずれた | 地上から高さ $h$ まで、加速度 $a$ の等加速度直線運動をする。その瞬間の速さは $v \= \\sqrt{2ah}$ であり、はずれた直後の「鉛直投げ上げ」の初速度となる。 |
| 綱からはずれた後の P の運動 | 綱からはずれた後は、重力のみが働く加速度 $-g$ の「鉛直投げ上げ運動」となる。最高点に達した後、地上（はずれた点から下方へ距離 $h$）まで落下する。 |
| 地上に落下するまでの時間 | 「引き上げ開始から $h$ に達するまでの時間 $t\_1$」と「はずれてから地上に落下するまでの時間 $t\_2$」を分けて計算し、その合計を求める必要がある。$t\_2$ は変位の式 $-h \= vt\_2 \- \\frac{1}{2}gt\_2^2$ から算出できる。 |

### 8 運動方程式

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| A（質量 $M$）と B（質量 $M/2$）を糸の両端に結び、滑車を通す | 糸は質量無視・非伸縮とする。A, B（およびC）の加速度の大きさ $a$ は等しく、糸の張力 $T$ は系全体で共通である。 |
| 滑らかな斜面 | 斜面と物体Aの間の摩擦は無視できる（摩擦力 $0$）。斜面に垂直な方向では力がつりあっている。 |

### 48 分子運動論

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 半径 $r$ の球形容器 | 幾何学的な対称性より、分子が壁に衝突する点は常に中心から距離 $r$ にある。また、容器の表面積 $S \= 4\\pi r^2$、体積 $V \= \\frac{4}{3}\\pi r^3$ を用いる。 |
| 弾性衝突をする | はねかえり係数 $e=1$。衝突前後で分子の速さ $v$ は変化せず、運動エネルギーが保存される。壁に平行な速度成分は変化せず、壁に垂直な成分のみが反転する。 |
| 分子どうしの衝突はない | 分子は次の壁面衝突まで等速度運動（直線運動）を続ける。気体分子の平均自由行程が容器のサイズより十分に長いことを前提としている。 |
| 入射角は $\\theta$ | 分子の速度ベクトルと壁の法線（中心を通る半径方向）とのなす角が $\\theta$ である。壁に垂直な速度成分（法線成分）の大きさは $v \\cos \\theta$ と表せる。 |
| 1回の衝突で与える力積 | 分子の運動量の変化 $\\Delta \\vec{p}$ に着目する。法線方向を正とすると、変化量は $m(-v \\cos \\theta) \- m(v \\cos \\theta) \= \-2mv \\cos \\theta$。壁が受ける力積はその反作用で大きさを問われているため $2mv \\cos \\theta$ となる。 |
| 次に衝突するまでに進む距離 | 円（球）の弦の長さに相当する。中心角を考えた二等辺三角形の幾何学的関係から、移動距離 $L \= 2r \\cos \\theta$ と導ける。 |
| 時間 $t$ の間に衝突する回数 | 衝突の間隔（周期） $\\Delta t \= \\frac{\\text{距離}}{\\text{速さ}} \= \\frac{2r \\cos \\theta}{v}$ を用いて、回数 $n \= \\frac{t}{\\Delta t} \= \\frac{vt}{2r \\cos \\theta}$ と計算できる。 |
| 気体分子全体が及ぼす力 | 1個の分子が及ぼす平均の力 $f$ は、（1回の力積）×（単位時間当たりの衝突回数）で求められる。$f \= (2mv \\cos \\theta) \\cdot \\frac{v}{2r \\cos \\theta} \= \\frac{mv^2}{r}$ となり、入射角 $\\theta$ に依存しない一定値になる。 |
| 気体の圧力 $P$ | 圧力の定義 $P \= \\frac{F}{S}$ より、全分子が及ぼす力 $F \= \\sum f$ を球の全表面積 $4\\pi r^2$ で割ることで導出する。 |
| 理想気体の状態方程式と比較 | $PV \= nRT \= \\frac{N}{N\_A}RT$。分子運動論から導いた $P$ の式と、熱力学的な状態方程式を等置することで、ミクロな運動量とマクロな温度 $T$ を結びつける。 |

### 49 分子運動論

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 断熱容器 | 外部との熱の出入りがない（$Q=0$）。第一法則 $Q \= \\Delta U \+ W\_{out}$ より、気体が外部へ仕事をする（$W\_{out} \> 0$）と、その分だけ内部エネルギーが減少（$\\Delta U \< 0$）し、温度が下がる。 |

| 単原子分子の理想気体 | 内部エネルギーが $U \= \\frac{3}{2}nRT \= \\frac{3}{2}PV$ と表せる。定積モル比熱 $C\_V \= \\frac{3}{2}R$、定圧モル比熱 $C\_P \= \\frac{5}{2}R$ である。 |

| ピストンがDに達して排水を終わる | 状態Ⅲにおいて、ピストンの上には水が全くない状態になる。よって状態Ⅲの気体の圧力は $p\_3 \= p\_0$ である。 |
| 気体が冷えるにしたがって（状態Ⅲ→Ⅳ） | ピストンの上には水がない状態（大気圧のみ）で下降する。したがって、この過程は **定圧変化**（$p\_3 \= p\_4 \= p\_0$）である。 |
| Cを開くと...注入され（状態Ⅳ→Ⅰ） | ピストンがCから元の位置まで下がる間に、再びピストンの上に水が溜まっていく。水深が $0$ から $b$ へと戻るため、気体の圧力 $P$ は増加する。 |

| 吸収する熱量 $Q$ | 熱力学第一法則 $Q \= \\Delta U \+ W$ を用いる。$\\Delta U \= \\frac{3}{2}(P\_{after}V\_{after} \- P\_{before}V\_{before})$ として計算可能。 |

### 62 波の性質

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |

| $x$ 軸の正の向きに進んでいる | 波形全体が時間の経過とともに右方向へ平行移動する。媒質の速度の向きを判定する際、波形をわずかに右にずらして作図（微小時間後の波形）することが不可欠。 |
| 図1（$y$-$x$ グラフ） | 波の「写真（瞬間の姿）」。隣り合う山から山の距離などから波長 $\\lambda \= 8$ cm、グラフの振れ幅から振幅 $A \= 3$ mm と読み取れる。 |

| 媒質の密度が最大（密） | 周囲の媒質がその点に向かって集まってくる状態。$y$-$x$ グラフにおいて、「左側の媒質が右にずれ（$y\>0$）、右側の媒質が左にずれている（$y\<0$）」位置、すなわち **$y=0$ かつグラフの傾きが負（右下がり）** の点が該当する。 |
| 密度が最大になるまで...あと何秒 | 「密」の状態（位相）が波の速さ $v$ で移動すると考える。現在 $x=11$ cm の地点の直後にある密の位置が、$x=11$ cm に到達するまでの距離 $\\Delta x$ を求め、$\\Delta t \= \\Delta x / v$ で計算する。 |
| 媒質の速度が 0 | 媒質の単振動における「端（折り返し点）」の状態。図1または図2において、変位 $y$ が最大または最小（山または谷）となっている位置。 |
| 媒質の速度が右向き最大 | 媒質が振動の中心（$y=0$）を正の向きに通過する状態。波が右に進む場合、波形を少し右にずらしたときに $y$ が増える場所、すなわち **$y=0$ かつグラフの傾きが負（右下がり）** の位置。 |
| 図2のようになる位置 | $t=0$ で $y=0$ であり、かつ直後に $y$ が減少（左に移動）し始める点。図1の $y=0$ の点のうち、波形を少し右にずらしたときに $y$ が負になる場所（**グラフの傾きが正＝右上がり**）を探す。 |
| 周期性（$x=100$ cmなど） | 波は一定の間隔（$\\lambda$ や $T$）で同じ状態を繰り返す。大きな数値は $\\lambda$ や $T$ で割り、余りの部分（位相）に着目することで、図1・図2の描画範囲内の情報に変換できる。 |

| $L\_2$ を $L\_1$ から何 cm 離せばよいか | $L\_1$ による像の位置（$L\_1$ から $b\_1$）と $L\_2$ の位置の差が、$L\_2$ にとっての物体距離 $a\_2$ となる。レンズ間距離を $d$ とすると、$a\_2 \= d \- b\_1$ が成り立つ（$b\_1 \> d$ ならば仮想物体として $a\_2 \< 0$ となる）。 |
| 凹レンズ $L\_3$（焦点距離 12 cm） | 凹レンズであるため、焦点距離は負の値 $f\_3 \= \-12$ cm として公式に代入する。 |

| $L\_1, L\_3$ による像 | 最終的な像の性質（実像・虚像）は、第2レンズ $L\_3$ についての公式から得られる $b\_3$ の符号で決定する。 |

### 75 光波

| ①問題文の記述 | ②読み取れる物理的な情報・条件 |
| :---- | :---- |
| 200枚の歯をもつ歯車 G | 歯車が1回転する間に、光は200回遮断され、200回通過する。つまり、1回転で200周期分のパルスが生じる（1周期 $a$ は「歯1枚分＋隙間1枚分」が通過する時間に相当する）。 |
| 光は...周期的に遮断する | 光が連続光ではなく、図2のような矩形波状のパルス光として伝播していることを示す。 |
| $AB$ は $BC$ に等しく、$AP\_1$ は $CP\_2$ に等しい | 検出器 $P\_1$ と $P\_2$ に届く光の経路差は、純粋に往復距離 $AB \+ BC \= 2l$ である。装置固有の微小な光路差（$A$ から $C$ など）は無視、または相殺されている。 |
| 図2：実線は $P\_1$ 、破線は $P\_2$ | $P\_1$ は基準となるパルス波形、$P\_2$ は距離 $2l$ を通過して遅れて届いたパルス波形を表す。 |
| 横軸は時間で 1目盛りは $5.0 \\times 10^{-6}$ \[s\] | 図中の長さ（目盛り数）にこの値を掛けることで、実際の時間経過 \[s\] に換算できる。 |
| 間隔 $a$（周期）は 4.0 目盛り | パルス1周期の時間 $T$ が $4.0 \\times 5.0 \\times 10^{-6}$ s である。これと歯数から歯車の回転速度（回転数）が計算できる。 |
| $b$ 目盛りだけずれていた | 光が距離 $2l$ を進むのに要した時間 $\\Delta t$ が、パルス周期 $a$ の整数倍を除いて $b$ だけズレていることを示す（$\\Delta t \= m \\cdot a \+ b$）。 |
| 水槽に水を満たす / 屈折率は 1.3 | 水槽内の光速が $V \= c / 1.3$ に低下する。これにより、水槽の長さ 500m を通過する時間が $500/c$ から $1.3 \\times 500/c$ へ増加し、時間差 $b$ が変化する。 |
| $a$ は徐々に伸びて $a'=5.0$ / $b$ は徐々に縮んで $b'=0.6$ | 歯車の回転をゆっくり変えたとき、光の往復時間 $\\Delta t$ 自体は不変だが、周期 $a$ が変わることで相対的なズレ $b$ が連続的に変化したことを意味する。 |
| $b$ が「徐々に」変化した | $\\Delta t \= m \\cdot a \+ b$ における整数 $m$ の値が、変化の過程でどう推移したかを特定するためのヒント。不連続な飛び（パルスの追い越し）がなかったかを確認できる。 |
| 光速 $c \= 3.0 \\times 10^8$ \[m/s\] | 既知の値として計算に用いる。媒質が空気（または真空）の部分の速さ。 |
